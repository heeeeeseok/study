## 편집 관련
- ### 라인 이동
      ctrl + shift + enter
- ### 영역 선택
      ctrl + w
- ### 세미콜론 자동완성
      ctrl + shift + enter
- ### 클래스명.매서드 출력
      soutm
- ### 줄 삭제
      ctrl + x
- ### 주석 처리
      ctrl + /

## 코드 탐색 관련
- ### 파일 또는 패키지에서 특정 코드 탐색
      ctrl + shift + f
- ### 선언된 파일로 이동
      ctrl + b
- ### 구현한 파일로 이동
      ctrl + alt + b
  
## 파일 이동 관련
- ### 작업 파일 히스토리
      ctrl + e

## 생성자, getter, setter
    alt + insert

## new 생성자 자동 선언
    ctrl + alt + v

## 테스트 코드 생성
    ctrl + shift + t
